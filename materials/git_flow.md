For this course I would ask you to have the next git flow

- Download the repo
- Create your own PRIVATE repository
- Add me to the users access list

When you are doing the JS/AT tasks follow the next flow

- Create a new branch
- Make a changes
- Upload the branch to the remote repo
- Create a PR. Optionally ask for review
- Merge with main branch

When you are doing the TAF tasks follow the next flow

- Create a new PRIVATE repo
- Setup one of the frameworks cypress or playwright. 1 repo for 1 TAF
- Start implementing the tasks one by one
- Create a new branch
- Make a changes
- Upload the branch to the remote repo
- Create a PR. Optionally ask for review
- Merge with main branch
